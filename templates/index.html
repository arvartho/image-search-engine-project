DWS Advanced DB Topics assignment

This is an image search engine based on histogram similarity, developed for the Advanced DB Topics cource of DWS Master program.

The database consists of celebrities headshot images selected randomly from the web. The algorithm detects the face in the image and divides the region around it in segements, then calculates the histogram of each segment and uses it as the descriptor vector. The engine will return the k most similar images according to the selected distance metric.
